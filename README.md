## fantasy-soccer-spa

- Hosted on [DO](https://fantasy-soccer-spa-htw4m.ondigitalocean.app/)

### How to run (the app)
- During development, do the following and visit `localhost:3000`
```bash
npm run start
```

- Run the following to get the production build (in `./build`)
```bash
npm run build
```
